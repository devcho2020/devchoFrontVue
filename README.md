# 🚀 Tech Stack & Features

Vue 3와 Vite를 기반으로 구축된 프론트엔드 프로젝트의 기술 스택 및 주요 구현 기능입니다.

## 🛠 Tech Stack (Versions)

| Category | Technology | Version | Description |
| :--- | :--- | :--- | :--- |
| **Core** | Vue.js | ^3.5.30 | Composition API 기반 모던 웹 프론트엔드 |
| **Build Tool** | Vite | ^8.0.1 | 초고속 빌드 및 개발 서버 환경 |
| **State** | Pinia | ^3.0.4 | 가볍고 직관적인 전역 상태 관리 |
| **Router** | Vue Router | ^4.6.4 | SPA 페이지 라우팅 및 네비게이션 관리 |
| **Network** | Axios | ^1.16.0 | RESTful API 비동기 통신 |
| **Styling** | Tailwind CSS | ^4.2.2 | 유틸리티 퍼스트 모던 CSS 프레임워크 |
| **Utility** | Day.js | ^1.11.20 | 가벼운 날짜 및 시간 데이터 파싱/포맷팅 |

---

## ✨ Key Features & Implementation

* **고성능 빌드 및 개발 환경 (Vite)**
    * ESM 기반의 빠른 HMR(Hot Module Replacement)을 활용한 쾌적한 개발 환경 구축
* **전역 상태 관리 (Pinia)**
    * 컴포넌트 간 복잡한 데이터 공유 및 모달, 유저 세션 등 전역 상태 효율적 관리
* **모던 UI 스타일링 (Tailwind CSS v4)**
    * 유틸리티 클래스를 활용한 반응형 웹 디자인 및 일관된 디자인 시스템 적용
* **비동기 API 통신 (Axios)**
    * 백엔드 서버와의 RESTful API 연동 및 데이터 송수신 처리
* **날짜 및 시간 처리 (Day.js)**
    * 일정, 로그, 만료일 등 애플리케이션 내 날짜/시간 데이터 가공 및 포맷팅
